记录麻雀对战点数并统计个人数据的应用。
方便查看玩家数据，系统分析自己的弱点在哪儿
方便实时查看点数，根据情况来决定策略。
日麻是一个打顺位的游戏~        

v1.0
1、和牌或流局后设置点数，可以实时查看各家数据，方便决定策略。
2、可以查看对战玩家立直率、和牌率等基本状况
3、可以查看历史顺位、平均顺位等。

v1.1
1、优化了自定义桌面占用内存，若不选择图片，则返回默认状态；
2、点数早见表由图改成了表格。
3、和牌后先选择立直玩家设置供托后才进入设置点数界面。
4、不用手输入点数了，选择番符后直接计算出来。
5、去除语音识别功能，减少权限的申请。

1、增加了反馈通道
